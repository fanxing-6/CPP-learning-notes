# C++三路比较运算符

*左操作数* `<=>` *右操作数*

表达式返回一个对象（其实是这个`std::strong_ordering`），使得

- 若 左操作数 < 右操作数 则 (a <=> b) < 0

- 若 左操作数 > 右操作数 则 (a <=> b) > 0

  [^1]: 这不就是相当于（左操作数 - 右操作数 即 a-b 吗）

  

- 而若 左操作数 和 右操作数 相等/等价则 (a <=> b) == 0。

```cpp
	int a{ 2 };
	int b{ 5 };
	if (a<=>b >0)
	{
		cout << "a>b"<< endl;
	}
	else if (a<=>b ==0)
	{
		cout << "a=b" << endl;
	}
	else
	{
		cout << "a<b" << endl;	
	}
```
